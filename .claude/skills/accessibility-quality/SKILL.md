# Accessibility And Quality Skill

Check static websites for practical quality.

Rules:
- Use semantic HTML.
- Use proper heading order.
- Buttons and links must have clear text.
- Colour contrast must be readable.
- Forms, if any, need labels.
- Images need useful alt text.
- Do not rely on colour alone to communicate meaning.
- Keyboard navigation should work.
- Keep page load light.
- Avoid unnecessary dependencies.
- Validate there are no broken internal links.

Acceptance checks:
- One H1 only.
- Sections use clear headings.
- CTA buttons are accessible.
- Page works without JavaScript for core content.
- No console errors.
