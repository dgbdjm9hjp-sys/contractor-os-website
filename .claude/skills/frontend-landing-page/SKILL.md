# Frontend Landing Page Skill

Build clean, fast, responsive landing pages using simple HTML, CSS, and JavaScript.

Rules:
- Start with the page goal.
- Keep one primary CTA.
- Use clear visual hierarchy.
- Put the main value above the fold.
- Use short sections.
- Use cards only where they improve scanning.
- Avoid clutter.
- Avoid generic SaaS styling.
- Avoid overdesigned animations.
- Optimise for mobile first.
- Keep file structure simple.

Preferred structure:
- Hero
- Problem
- Offer
- What is reviewed
- What the buyer gets
- Trust or guardrail section
- Who it is for
- How it works
- Final CTA

Quality bar:
The page should look like a credible UK B2B service website, not a developer demo.
