# Responsive Design Skill

Build layouts that work from mobile to desktop.

Rules:
- Design mobile first.
- Use flexible grids.
- Avoid fixed widths that break on small screens.
- Keep line lengths readable.
- Make buttons easy to tap.
- Stack cards cleanly on mobile.
- Use consistent spacing.
- Test common widths: 375px, 768px, 1024px, 1440px.
- Do not hide important content on mobile.
- Keep performance light.

Acceptance checks:
- No horizontal scrolling.
- CTA visible and usable on mobile.
- Text is readable without zooming.
- Cards stack cleanly.
- Header does not crowd the hero.
